Note: Follow these steps To run the application -

Step-1:
Go to "rad-diff-checker-frontend" folder by using "cd command" then run the command
"npm i" then after "npm run dev", the app will be running on "https://localhost:3000".

Step-2:
Open "rad-diff-checker-Backend" folder by using "cd command" then run the command
"npm i" then after "node server/app.js"
